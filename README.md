# Kotlin

Kotlin 是一个基于 `JVM` 的新的编程语言，由 JetBrains 开发   
Kotlin 可以编译成`Java字节码`，也可以编译成`JavaScript`，方便在没有JVM的设备上运行    

- 比Java更安全
- 比Java更简洁

- 增强版（简洁版）的`Java`，Android版的Swift😅
